Politique de Confidentialité

Dernière mise à jour : 1er décembre 2025

1. Collecte des données

Nous collectons les informations suivantes :

  • Informations de compte : Email, nom, photo de profil
  • Contenu créé : Messages, noms, réponses personnalisées
  • Données d'utilisation : Vues, interactions, temps passé
  • Données techniques : Adresse IP, navigateur, appareil

2. Utilisation des données

Vos données sont utilisées pour :

  • Fournir et améliorer nos services
  • Personnaliser votre expérience utilisateur
  • Analyser l'utilisation du service
  • Communiquer avec vous (notifications, support)

3. Partage des données

Nous ne vendons pas vos données. Nous les partageons uniquement :

  • Avec votre consentement explicite
  • Avec nos fournisseurs de services (hébergement, email)
  • Pour respecter des obligations légales

4. Protection des données

Nous protégeons vos données avec :

  • Chiffrement SSL/TLS
  • Authentification sécurisée Firebase
  • Contrôles d'accès stricts
  • Audits de sécurité réguliers

5. Vos droits

Vous avez le droit de :

  • Accéder à vos données personnelles
  • Corriger des données inexactes
  • Supprimer vos données
  • Exporter vos données
  • Vous opposer au traitement

6. Conservation des données

Nous conservons vos données :

  • Tant que votre compte est actif
  • Jusqu'à 2 ans après inactivité
  • Selon les obligations légales

7. Contact

Pour exercer vos droits ou poser des questions :

Email : privacy@lovecraft.com

Adresse : LoveCraft, Porto-Novo, Bénin
